# GitJob KZ - IT Job Board для Казахстана
GitJob - платформа для поиска IT-вакансий в Казахстане с карьерным тестом и рекомендациями. Фокус - быстрый обзор рынка, удобный поиск и персональные подсказки.

## Ключевые возможности
- Каталог IT-вакансий с поиском, фильтрами и сортировкой.
- Страницы вакансий с подробным описанием и навыками.
- Каталог компаний и страницы компаний с активными вакансиями.
- Карьерный тест с итоговыми рекомендациями по роли и этапу развития.
- Персональные рекомендации вакансий на основе профиля.
- Обучающий гид по этапам создания IT-продукта.

## Технологический стек
- React, Vite, Tailwind CSS
- FastAPI, SQLAlchemy, Alembic
- PostgreSQL, Redis
- Docker, Docker Compose, Nginx

## Быстрый старт (Docker)
### Требования
- Docker
- Docker Compose
- Файл `.env` на основе `.env.example`

### Запуск
```bash
docker compose up --build
```

### Проверка
- `http://localhost` - фронтенд
- `http://localhost/healthz` - health check
- `http://localhost:8000/docs` - Swagger

## Переменные окружения
- `POSTGRES_DB`
- `POSTGRES_USER`
- `POSTGRES_PASSWORD`
- `DATABASE_URL`
- `JWT_SECRET_KEY`
- `INTERNAL_SECRET`
- `REDIS_URL`

## Жизненный цикл разработки
- Discovery: исследование рынка и формулировка задачи.
- Design: проектирование API, данных и интерфейса.
- Build: разработка фронтенда, бэкенда и интеграций.
- Verify: тестирование ключевых сценариев.
- Release: контейнеризация и публикация.
- Maintain: поддержка, мониторинг, обновления.

## Статус проекта
- MVP готов.
- Проект предназначен для demo / портфолио.
